# Unlim8ted Boot Filesystem

This directory contains bootloader configuration and the staged custom kernel
payloads used by `../build/build.sh`.

Required kernel payloads:

- `./kernel/x86_64/vmlinuz-unlim8ted`
- `./kernel/x86_64/initramfs-unlim8ted.img`
- `./kernel/arm64/Image`
- `./kernel/arm64/initramfs-unlim8ted.img`
